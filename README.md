# Final Project: Treact Landing Page

## 📌 Description
This project is a landing page built with **HTML, CSS, and JavaScript**, focused on clean design, clear structure, and responsive behavior.  
The main goal was to **replicate a modern interface inspired by Treact**, applying best practices in layout, component organization, and precise styling to achieve a visually consistent and professional result.

The project was built entirely by hand, without frameworks, to strengthen frontend fundamentals and maintain full control over the layout.

---

## 🚀 Features
- **Fully responsive design** for desktop, tablet, and mobile.  
- **Functional navbar** with a mobile hamburger menu.  
- **Structured sections**: Hero, Features, Quality, Steps, Values, Pricing, Testimonials, Start, and Footer.  
- **Organized CSS** with a focus on spacing, typography, and visual hierarchy.  
- **Clean and maintainable code**, using semantic HTML and clear class naming.  

---

## 🛠️ Installation
1. Clone this repository:  
   `git clone https://github.com/magdalenabatres/final-project.git`
2. Navigate into the project folder:  
   `cd final-project`
3. Open the `index.html` file in your browser.

No dependencies or additional installation required.

---

## 📖 Usage
Simply open the `index.html` file in any modern browser.  
The page will load all sections and styles automatically.
